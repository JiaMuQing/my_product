Chromium · Manifest V3

Account Switch

Cookie 配置 · 多账号快照与一键切换

基于 Chromium 的浏览器扩展：按站点保存 Cookie 快照，支持自定义名称与分组。切换账号时会清除当前 URL 作用域下的 Cookie，并恢复已保存的集合。数据默认仅保存在本机。

核心能力

  • 按站点管理多个账号（profile），分组与命名清晰
  • 切换时自动清理作用域内 Cookie 并写入快照，避免串号
  • 界面语言跟随浏览器；支持英文与简体中文等 i18n

免费版限制可保存的账号数量（可在源码中配置 FREE_MAX_PROFILES，默认 5）。

专业版通过服务端签发的 JWT 许可证解锁（扩展内仅嵌入公钥校验；私钥留在服务器，常见流程：官网 Stripe 收款 → Webhook 签发 JWT）。

权限与隐私

  • cookies、storage、tabs：读写 Cookie、本地存储与标签页上下文
  • 按需申请站点或全网访问权限，仅在保存/切换时使用

构建：npm install 与 npm run build，在 chrome://extensions 中加载 dist/。完整说明、本地 JWT 测试与隐私政策见仓库文档（如 docs/PRIVACY.md）。

Chromium · Manifest V3

Account Switch

Cookie profiles · Snapshots & one-click switching

A Chromium extension that saves Cookie snapshots per site, with custom names and groups. When you switch accounts, it clears cookies for the current URL scope and restores the saved set. Data stays on your device by default.

Key features

  • Manage multiple accounts (profiles) per site with clear naming and groups
  • On switch, clears in-scope cookies and applies the snapshot to prevent cross-account leakage
  • UI follows the browser language; English and Simplified Chinese i18n supported

The free tier limits how many accounts you can save (configure FREE_MAX_PROFILES in source, default 5).

Pro unlocks via a server-signed JWT license (the extension embeds only the public key; the private key stays on the server—typical flow: Stripe on the website → webhook issues JWT).

Permissions & privacy

  • cookies, storage, tabs: read/write cookies, local storage, and tab context
  • Host permissions are requested as needed (per-site or broader), only for save/switch flows

Build: npm install and npm run build, then load dist/ from chrome://extensions. Full docs, local JWT testing, and the privacy policy are in the repository (e.g. docs/PRIVACY.md).
